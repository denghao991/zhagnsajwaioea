# 可用性检查功能说明

## 概述
可用性检查用于检测云服务资源的健康状态，支持 CSS、OBS、ECS、RDS 等多种云服务。
当用户发起可用性检查请求后，系统会对目标服务进行连通性测试、状态码验证、响应时间检测。

## 可用性检查结果说明
检查结果分为：正常、异常、超时三种状态。
如果检查项显示异常，请先确认资源是否在运行中。

## CSS 云搜索服务
CSS（Cloud Search Service）提供全文搜索能力。
如果 CSS 资源信息展示不全，请联系 CSS 产品团队修改展示逻辑。

## OBS 对象存储
OBS（Object Storage Service）提供海量数据存储。
如果 OBS 未对接 OA 系统，需要联系 OBS 产品团队处理。

## ECS 云服务器
ECS 提供弹性计算能力。

## 容量优化
容量优化功能可以分析云资源使用率，给出扩缩容建议。
如果半天不出结果，请提交工单联系 oncall 人员确认。

## 月报工单
月报工单汇总当月云资源使用情况。
如果信息不全，请尝试重新生成；仍不行再联系 oncall。

## 权限说明
可用性检查功能需要具备对应云服务的只读权限。
如果没有权限，请联系主账号授权。
